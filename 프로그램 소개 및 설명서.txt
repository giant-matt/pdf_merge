========================================
PDF 변환 & 취합 프로그램
========================================

[ 이 프로그램은? ]

- Why (왜 만들었나?)
    - 증빙자료 제출 시 여러 형식의 파일(PDF, 이미지, 문서)을 하나의 PDF로 취합하는 작업이 번거롭고 시간이 많이 소요됨

- How (어떻게 해결하나?)
    - GUI 기반으로 폴더를 선택하면 지원 파일을 자동 인식하고, 순서 조정 후 한 번에 병합
    - 이미지/문서 파일은 자동으로 PDF로 변환 후 병합

- What (무엇을 하나?)
    - 다양한 형식의 파일을 하나의 PDF로 자동 변환 및 병합하는 데스크톱 프로그램

========================================

[ 핵심 기능 ]

1. 다양한 파일 형식 지원
    - PDF (.pdf)
    - 이미지 (.jpg, .jpeg, .png)
    - MS Office (.doc, .docx, .xls, .xlsx, .ppt, .pptx)
    - 한글 문서 (.hwp, .hwpx)

2. 자동 PDF 변환
    - 이미지 파일 → PDF 변환
    - Office 파일 → PDF 변환
    - 한글 파일 → PDF 변환

3. 파일 순서 조정
    - ▲▼ 버튼으로 순서 변경
    - ✕ 버튼으로 목록에서 제외

4. 진행 상황 표시
    - 프로그레스 바로 변환/병합 진행률 표시
    - 로그 영역에 상세 진행 내역 출력

5. 결과 요약
    - 성공/실패 파일 목록 표시
    - 실패 원인 상세 표시

========================================

[ 사용 방법 ]

1. 프로그램을 실행합니다.
2. '찾아보기' 버튼을 클릭하여 병합할 파일이 있는 폴더를 선택합니다.
3. 파일 목록에서 순서를 조정하거나 제외할 파일을 선택합니다.
4. '병합 시작' 버튼을 클릭합니다.
5. 병합된 PDF 파일이 선택한 폴더에 저장됩니다.

========================================

[ 주의 사항 ]

- MS Office 파일 변환을 위해 Office가 설치되어 있어야 합니다.
- 한글 문서 변환을 위해 한/글 프로그램이 설치되어 있어야 합니다.
- Office 2016, O365 모두 지원됩니다.
