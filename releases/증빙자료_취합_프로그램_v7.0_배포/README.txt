===============================================
   PDF & 이미지 병합 프로그램
===============================================

프로그램 버전: 5.1
제작일: 2025

===============================================

📦 설치 방법

1. zip 파일의 압축을 원하는 위치에 풀어주세요

2. 폴더 안의 "PDF_Image_Merger_v5.1.exe" 파일을 더블클릭하여 실행합니다

3. Python 설치가 필요 없습니다!

📌 v5.1 새로운 기능:
   - 한글 파일 변환 안정성 개선
   - 자세한 변환 성공/실패 로그 표시
   - 문서 변환 오류 처리 개선

📌 v5.0 기능:
   - 실시간 진행률 표시 (진행 바 + 퍼센트 %)
   - 단계별 작업 상태 표시 (변환 중, 병합 중 등)
   - 개선된 파일 목록 표시 및 오류 처리

📌 v4.0 기능:
   - Word 파일 (.docx, .doc) 자동 PDF 변환 지원
   - 한글 파일 (.hwp, .hwpx) 자동 PDF 변환 지원
   ※ Word/한글 변환 기능 사용 시 해당 프로그램이 설치되어 있어야 합니다

⚠️ Windows 보안 경고가 나타나는 경우:
   - "Windows의 PC 보호" 창이 나타나면 "추가 정보"를 클릭
   - 그 다음 "실행" 버튼을 클릭하여 실행하세요
   - 이 프로그램은 안전하며, 디지털 서명이 없어 경고가 표시됩니다

===============================================

🚀 빠른 시작

1. 프로그램 실행
2. "찾아보기" 버튼으로 병합할 파일이 있는 폴더 선택
3. 파일 순서 확인 및 조정
4. "병합 시작" 버튼 클릭
5. 완료!

자세한 사용법은 "사용설명서.txt"를 참고하세요.

===============================================

💻 시스템 요구사항

- Windows 10 이상
- 최소 100MB 여유 공간
