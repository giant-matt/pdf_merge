📌 프로그램 소개
--------------------------------------------------------------------------------
여러 형식의 증빙자료 파일들을 하나의 PDF 파일로 자동 병합하는 프로그램입니다.
이미지, PDF, 워드 문서, 한글 문서를 하나의 파일로 손쉽게 통합할 수 있습니다.


✅ 주요 기능
--------------------------------------------------------------------------------
• PDF, 이미지, 워드, 한글 파일을 하나의 PDF로 병합
• 파일 순서를 자유롭게 조정 가능 (▲▼ 버튼)
• 원하지 않는 파일 제외 가능 (✕ 버튼)
• 파일 변환 및 병합 결과를 실시간 로그로 확인
• 성공/실패 파일 목록을 상세하게 표시


📁 지원 파일 형식
--------------------------------------------------------------------------------
• PDF 파일 (.pdf)
• 이미지 파일 (.jpg, .jpeg, .png)
• 워드 문서 (.doc, .docx)
• 한글 문서 (.hwp, .hwpx)


🚀 사용 방법
--------------------------------------------------------------------------------
1. 프로그램 실행
   - "증빙자료_취합_프로그램_v6.0.exe" 더블클릭

2. 폴더 선택
   - "찾아보기" 버튼을 클릭하여 병합할 파일들이 있는 폴더 선택
   - 자동으로 지원하는 파일 형식만 목록에 표시됩니다

3. 파일 순서 조정 (선택사항)
   - 파일을 클릭하여 선택
   - ▲ 버튼: 선택한 파일을 위로 이동
   - ▼ 버튼: 선택한 파일을 아래로 이동
   - ✕ 버튼: 선택한 파일을 병합 목록에서 제외 (파일은 삭제되지 않음)

4. 병합 시작
   - "병합 시작" 버튼 클릭
   - 진행 상황은 하단의 "진행 상황" 영역에서 확인 가능

5. 결과 확인
   - 병합이 완료되면 결과 메시지가 표시됩니다
   - 성공/실패 파일 목록이 표시됩니다
   - 병합된 PDF 파일은 선택한 폴더에 "[폴더명]_merged.pdf" 이름으로 저장됩니다


⚙️ 시스템 요구사항
--------------------------------------------------------------------------------
• 운영체제: Windows 10/11
• 한글 문서 변환 시: 한글과컴퓨터 한/글 프로그램 설치 필요
• 워드 문서 변환 시: Microsoft Word 설치 필요
• 이미지 및 PDF 병합: 추가 프로그램 설치 불필요


⚠️ 주의사항
--------------------------------------------------------------------------------
• 한글 문서를 PDF로 변환하려면 컴퓨터에 한/글 프로그램이 설치되어 있어야 합니다
• 워드 문서를 PDF로 변환하려면 MS Word가 설치되어 있어야 합니다
• 변환 실패한 파일은 자동으로 병합에서 제외되며 상세 내역이 로그에 표시됩니다
• 원본 파일은 절대 삭제되지 않습니다
• 병합 중에는 한/글이나 Word 프로그램을 실행하지 마세요


💡 문제 해결
--------------------------------------------------------------------------------
1. 한글 파일이 병합에 포함되지 않는 경우
   - 한/글 프로그램이 설치되어 있는지 확인
   - 진행 상황 로그에서 자세한 에러 메시지 확인
   - 한글 파일을 한/글에서 직접 열어 정상적인지 확인

2. 워드 파일이 병합에 포함되지 않는 경우
   - Microsoft Word가 설치되어 있는지 확인
   - 워드 파일이 암호로 보호되어 있지 않은지 확인

3. 프로그램이 실행되지 않는 경우
   - Windows Defender에서 차단되었을 수 있음
   - 마우스 우클릭 → "관리자 권한으로 실행"

4. 병합된 PDF가 너무 큰 경우
   - 이미지 파일이 고해상도일 경우 파일 크기가 클 수 있습니다
   - 필요시 이미지 파일을 미리 압축하여 사용하세요
