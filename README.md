### 💡 이 프로그램은?

- Why
    - 증빙자료 제출 시 여러 형식의 파일(PDF, 이미지, 문서)을 하나의 PDF로 취합하는 작업이 번거롭고 시간이 많이 소요됨
- How
    - GUI 기반으로 폴더를 선택하면 지원 파일을 자동 인식하고, 순서 조정 후 한 번에 병합
    - 이미지/문서 파일은 자동으로 PDF로 변환 후 병합
- What
    - 다양한 형식의 파일을 하나의 PDF로 자동 변환 및 병합하는 데스크톱 프로그램

---

### 🚀 핵심 기능

- [x]  **다양한 파일 형식 지원**
    - PDF (.pdf)
    - 이미지 (.jpg, .jpeg, .png)
    - MS Office (.doc, .docx, .xls, .xlsx, .ppt, .pptx)
    - 한글 문서 (.hwp, .hwpx)
- [x]  **자동 PDF 변환**
    - 이미지 파일 → PDF 변환 (Pillow 사용)
    - Office 파일 → PDF 변환 (win32com 사용)
    - 한글 파일 → PDF 변환 (HWP API 사용)
- [x]  **파일 순서 조정**
    - 드래그 없이 ▲▼ 버튼으로 순서 변경
    - ✕ 버튼으로 목록에서 제외
- [x]  **진행 상황 표시**
    - 프로그레스 바로 변환/병합 진행률 표시
    - 로그 영역에 상세 진행 내역 출력
- [x]  **결과 요약**
    - 성공/실패 파일 목록 표시
    - 실패 원인 상세 표시

---

### 🎯 타겟 대상

- 회계/재무팀 등 증빙자료를 자주 취합하는 업무 담당자
- 여러 형식의 문서를 하나의 PDF로 정리해야 하는 사무직 종사자

---

### 📝 개발 메모

- MS Office 파일 변환을 위해 Office가 설치되어 있어야 함
- 한글 문서 변환을 위해 한/글 프로그램이 설치되어 있어야 함
- win32com 사용 시 pythoncom.CoInitialize() 필요 (멀티스레드 환경)
- 파일 정렬은 자연 정렬(natural sort) 적용 (1, 2, 10 순서 보장)
